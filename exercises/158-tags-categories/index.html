Tags - Categories

Organize tags with categories

Learn Map with Set values for categorized tag systems

// Create system
const system = createCategorySystem();

// Add categories
addCategory(system, 'frontend');
addCategory(system, 'backend');

// Add tags to categories
addTagToCategory(system, 'frontend', 'javascript');
addTagToCategory(system, 'frontend', 'css');

// Get category tags
getCategoryTags(system, 'frontend');  // ['css', 'javascript']
