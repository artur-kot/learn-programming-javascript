🎨 GitHub User Display

Master DOM manipulation by displaying beautiful GitHub user profiles

1. Complete Profile Card

Create a complete profile card component that displays all user information in a visually appealing format. This demonstrates combining multiple DOM elements into a cohesive component.

2. Statistics Formatting

Learn how to format large numbers for display. Numbers like 1000+ become "1K", 1000000+ become "1M". This improves readability.

3. Profile Header Component

The header component includes avatar, display name, and username. It's the top portion of the profile card.

4. Bio Section

The bio section displays user information with graceful handling of missing data. Shows placeholder text when bio is not available.

5. Loading State

Show a loading indicator while fetching user data. This provides feedback to users that something is happening.

6. Error State

Display user-friendly error messages when something goes wrong. Use distinct styling to alert users to problems.

7. Multiple Profiles Grid

Display multiple user profiles in a responsive grid. This demonstrates rendering a collection of components.

8. Interactive Profile

Add click handlers to profiles for interactive features. Demonstrate event listeners and data attributes.
