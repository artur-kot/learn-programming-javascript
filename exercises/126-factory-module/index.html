Understanding the Module Pattern

What is the Module Pattern?

The module pattern is a way to organize and encapsulate code using closures and IIFE (Immediately Invoked Function Expression). It creates a private scope and reveals only what you want to be public.

// Module Pattern Structure const MyModule = (() => { // Private variables - only accessible inside the module let privateData = 'secret'; const privateArray = []; // Private functions function privateHelper() { return 'only inside'; } // Public interface (revealed) return { publicMethod() { return privateData; }, anotherPublic() { return privateHelper(); } }; })(); // Usage console.log(MyModule.publicMethod()); // ‚úÖ Works // console.log(MyModule.privateData); // ‚ùå undefined // console.log(MyModule.privateHelper()); // ‚ùå undefined
