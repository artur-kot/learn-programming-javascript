About This Exercise

This is a basic autocomplete setup demonstrating:

  • Backend: Fastify API server on port 3000
  • Frontend: Vite-bundled JavaScript application
  • Communication: REST API via fetch() with GET /search endpoint
  • Data: 250+ countries with name, code, and region

How It Works

  1. Type in the search box
  2. JavaScript listens for input changes
  3. Fetch request sends query to backend
  4. Fastify searches countries dataset
  5. Results returned as JSON
  6. Frontend displays results in real-time
